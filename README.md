# This is XeroLinux's Layan Rice

### Disclaimer

I would like to clarify one very important aspect of ricing. We use custom themes, and those might not get updated very often, if at all by their respective developers/artists, which might result in some parts of the system and apps not to get themed correctly. That, my friends, is unfortunately out of our hands.

### Supported Distros

The installer auto-detects your distribution and works on both **Arch Linux** and **Fedora KDE**.

### Fedora Notes

Before running the installer on Fedora KDE, make sure the **RPM Fusion** repositories (both **free** and **nonfree**) are enabled. Some packages and codecs depend on them. See the [RPM Fusion setup guide](https://rpmfusion.org/Configuration).

### Installation

To apply this rice on either **ArchLinux** or **Fedora** (Script will auto-detect), follow this :

```bash
git clone https://github.com/xerolinux/xero-layan-git
cd xero-layan-git/ && ./install.sh
```
